# Full Stack Library Application

A modern web application that simulates a library borrowing system, allowing users to browse, borrow, and return books.

## Features

- Search books by title
- Secure authentication and authorization
- Borrow and return books
- Pagination
- Sorting books by title (ascending/descending)

## Live Demo

Experience the application: [Library Application](https://angular-library-app.vercel.app/)

## Tech Stack

### Frontend

- Angular 21
- TypeScript
- Tailwind CSS

### Backend

- Java 21
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
